== Description ==

Angry Bunny Security Scanner is a powerful security plugin that helps protect your WordPress site from vulnerabilities, malware, and security threats. It performs regular security scans and provides actionable recommendations to improve your site's security.

= Key Features =

* Daily security scans
* Vulnerability detection
* Security score monitoring
* Email notifications for security issues
* Detailed security reports
* User-friendly interface

= Pro Features =

* Real-time security monitoring
* Advanced firewall protection
* Two-factor authentication
* Priority support
* Multi-site support
* Advanced malware scanning
* Custom scan scheduling
* And more!

= License Management =

The Pro version uses a license key system for activation. Each license key:
* Can be used on a specified number of sites
* Includes 1 year of updates and support
* Can be renewed annually
* Can be managed through your account dashboard

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/angry-bunny` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Angry Bunny Security screen to configure the plugin
4. Set your preferred scan frequency and notification settings

For Pro version:
1. Purchase a license key from our website
2. Install and activate the pro version
3. Enter your license key in the License Management section
4. Enjoy pro features!

== Frequently Asked Questions ==

= How often does the plugin scan my site? =

You can configure the scan frequency to run daily, weekly, or monthly. Pro users get access to custom scheduling options.

= What PHP version do I need? =

The plugin requires PHP 7.2 or higher.

= Is this plugin free? =

Yes, the basic version is completely free. A Pro version with advanced features is available for purchase.

= How does the licensing work? =

The Pro version requires a valid license key for activation. Each license key:
* Is valid for a specific number of sites
* Includes one year of updates and support
* Can be renewed annually
* Can be deactivated and moved to different sites

= Can I upgrade from free to pro? =

Yes! You can upgrade at any time. All your existing settings and scan history will be preserved.
